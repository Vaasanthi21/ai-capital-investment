# AI Capital Investment - Wealth Management System (React + Vite)

A modern, high-fidelity React web application implementing a state-of-the-art wealth management system matching the premium neon green & gold aesthetics from the reference mockup.

## Features
- **Interactive Landing Page**: Centered responsive details matching the user's reference mockup.
- **Glowing Hero Chart**: Custom animated canvas-based trading grid showing compounding growth with high-tech particles.
- **Glassmorphic Login Modal**: Custom inputs, Google/Apple OAuth buttons, and validations.
- **Glassmorphic Sign Up Modal**: Custom inputs for name, email, investment amount, target goals, and risk tolerance selectors.
- **Interactive Analytical Dashboard**: Displays upon sign-in/registration, containing:
  - Compounding Investment Growth projections line chart.
  - Dynamically recalculating Asset Allocation doughnut charts.
  - Dynamic holdings distribution table.
  - Interactive simulator sliders: Adjust initial capital, growth years, and risk profiles in real-time.
  - Live mock price feeds ticker (BTC, SPY, GLD).
  - AI Advisor monitoring and recommendation systems.
- **Advanced Dashboard Animations**:
  - Sidebar slide-in and widget cascade entry.
  - 60fps dynamic count-up value tickers.
  - Live price tick flashes.
  - Smooth tab swapping transitions.

## Tech Stack
- **Framework**: React.js
- **Bundler**: Vite
- **Charts**: Chart.js & React-Chartjs-2
- **Icons**: Lucide React
- **Styles**: Custom CSS3 variables, HSL color tokens, dark glassmorphism, glowing micro-animations, keyframes, scroll reveal triggers.

## How to Run

1. Navigate to the project directory:
   ```bash
   cd c:/Users/vasan/.gemini/antigravity/scratch/ai-capital-investment
   ```
2. Install npm dependencies (if not done already):
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
4. Open the displayed local address in your browser (usually `http://localhost:5173`).
